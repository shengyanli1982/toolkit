# httptool

**httptool** is a simple receiving HTTP responses.

## Installation

```bash
go get github.com/shengyanli1982/toolkit/pkg/httptool
```

## Explain

-   `HttpResponseItemsTotal`: The total number of items in the response.
-   `HttpResponseItemsID`: The ID of the item in the response.
-   `HttpQueryPaginated`: The query parameters for paginated requests.
-   `HttpResponsePaginated`: The paginated response.
-   `BaseHttpResponse`: The base response structure.
